Border Radius


Syntax: border-Radius:;

Example: border-Radius:;


For display round corners Border Radius property is used

IE9 and Opera use the same code. For Firefox a prefix "-moz-" needs to be added before the code. And for Webkit a prefix Browser "-Webkit-" needs to be added before the code.


Box-shadow property for all the browsers


For IE and Opera box-shadow:0px 30px 70px 20px;
For Mozilla Firefox -moz-box-shadow:0px 30px 70px 20px;
For Webkit Browsers -webkit-box-shadow:0px 30px 70px 20px;

No Border Radius (0px) has been applied to the top-left (green-pink) corner

Border Radius (30px) has been applied to the top-right (green-yellow) corner

Border Radius (70px) has been applied to the bottom-left (blue-yellow) corner

Border Radius (20px) has been applied to the bottom-left (blue-pink) corner
